Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After asemebling the dataset, we used Matplotlib to plot various aspects of the weather vs. latitude. Factors was looked at included: temperture, cloudiness,wind speed, and humudity. This site provides the source data and visulaizations created as part of the analysis, as well as explanations and descriptions of any trends and correlation witnessed.
